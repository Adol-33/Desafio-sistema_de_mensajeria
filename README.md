# Me SPA

https://adol-33.github.io/Desafio-sistema_de_mensajeria/

Proyecto de práctica para el curso de CSS Avanzado en Desafío LATAM.

## Descripción
Este proyecto simula una interfaz de chat tipo SPA (Single Page Application) con un diseño moderno y adaptable. Se enfoca en la aplicación de conceptos avanzados de CSS, como variables, flexbox y media queries.

## Estructura del proyecto
- `index.html`: Página principal con la estructura del chat.
- `assets/css/style.css`: Hoja de estilos principal.
- `assets/img/`: Imágenes de los usuarios y elementos gráficos.

## Características principales
- Diseño responsive y moderno.
- Uso de variables CSS para colores.
- Integración de Google Fonts y Font Awesome.
- Avatares de usuarios y botón de acción destacado.

## Instalación y uso
1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador.
3. Explora la interfaz y revisa el código para aprender sobre CSS avanzado.
